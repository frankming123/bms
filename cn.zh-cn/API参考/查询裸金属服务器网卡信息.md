# 查询裸金属服务器网卡信息<a name="ZH-CN_TOPIC_0131036398"></a>

## 功能介绍<a name="section798482464816"></a>

查询裸金属服务器的网卡信息，比如网卡的IP地址、MAC地址。

## URI<a name="section641994764720"></a>

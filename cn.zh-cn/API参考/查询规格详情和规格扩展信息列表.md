# 查询规格详情和规格扩展信息列表<a name="ZH-CN_TOPIC_0131326852"></a>

## 功能介绍<a name="section19330484"></a>

查询裸金属服务器的规格详情和规格的扩展信息。您可以调用此接口查询“baremetal:extBootType”参数取值，以确认某个规格是否支持快速发放。

## URI<a name="section39756633"></a>

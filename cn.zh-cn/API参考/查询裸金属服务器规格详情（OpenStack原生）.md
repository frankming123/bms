# 查询裸金属服务器规格详情（OpenStack原生）<a name="ZH-CN_TOPIC_0053158674"></a>

## 功能介绍<a name="section64833508"></a>

根据裸金属服务器的规格ID，查询规格的详细信息，比如规格名称、CPU核数、内存大小等。

## URI<a name="section46630661"></a>

GET /v2.1/\{project\_id\}/flavors/\{flavor\_id\}

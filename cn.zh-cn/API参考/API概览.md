## 接口介绍<a name="section1852025323614"></a>

裸金属服务器所提供的接口分为BMS接口与OpenStack原生接口。

通过配合使用BMS服务提供的接口和OpenStack原生接口，您可以完整地使用裸金属服务器的所有功能。例如创建裸金属服务器实例，可以使用OpenStack原生接口，也可以使用BMS接口进行创建。
